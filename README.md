行政書士（office-A）

## 概要
建設業界の雇用主が抱える「深刻な人手不足」と「複雑な在留資格手続き」という課題を解決するための、**BtoB特化型ランディングページ（LP）**です。
「共感・課題提示・解決策・信頼構築」の4ステップを軸に、法的な堅実さと、多忙な経営者が直感的に理解できる情報の整理を両立させました。

## 制作意図（課題解決アプローチ）
1. マーケティング視点に基づいた構成設計
- ターゲット: 建設業を営む30代〜60代前半の経営者・雇用主。
- 戦略: 広告流入を想定し、ヒートマップ分析による離脱ポイントを予測したCTA（行動喚起）配置を行いました。また、専門用語を噛み砕き、経営者が直面している「現場の悩み」にフォーカスした共感セクションを厚く設計しています。

2. 士業としての「信頼」を勝ち取るビジュアル設計
- 視覚戦略: 行政書士という職業柄、誠実さと清潔感を感じさせるネイビーとホワイトを基調とした配色を採用。過度な装飾を排し、情報の階層（Hierarchy）を明確にすることで、法的サービスの「正確さ」と「安心感」を視覚的に表現しました。

3. モバイル環境での利便性追求
- 解決策: 現場や移動中にスマホで閲覧する経営者を想定し、タップしやすいボタンサイズや、Q&Aセクションのアコーディオン実装など、限られた画面スペースで効率的に情報を取得できるUIを追求しました。

---

## 工夫した点
- 離脱防止設計: ユーザーが疑問に思うタイミングで「Q&A」や「料金案内」を配置。納得感を積み上げながら最下部のフォームへ導く導線を作成しました。

-Vanilla JSによる軽量実装: 外部ライブラリに頼らずJavaScriptを記述することで、読み込み速度を最適化。BtoBサイトに求められるストレスのない操作感を実現しています。

- WordPressではプラグインを追加しフォームを設定しました。
